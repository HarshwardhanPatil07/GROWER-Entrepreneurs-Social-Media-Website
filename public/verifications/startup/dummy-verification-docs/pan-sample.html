Sample PAN Card for Testing

This is a dummy PAN (Permanent Account Number) Card template that you can save as PDF to use for testing the verification system.

Note: This is not a real PAN card. In a real scenario, you would upload your actual PAN card.

How to use:

  1. Press Ctrl+P (or Command+P on Mac) to open the print dialog
  2. Select "Save as PDF" as the printer
  3. Save the file as "YourName-PAN-Card.pdf"
  4. Use this PDF to test the verification upload system

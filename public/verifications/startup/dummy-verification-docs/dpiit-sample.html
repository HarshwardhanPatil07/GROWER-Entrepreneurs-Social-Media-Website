Sample DPIIT Certificate for Testing

This is a dummy DPIIT (Department for Promotion of Industry and Internal Trade) Recognition Certificate template that you can save as PDF to use for testing the verification system.

Note: This is not a real certificate. In a real scenario, you would upload your actual DPIIT recognition certificate.

How to use:

  1. Press Ctrl+P (or Command+P on Mac) to open the print dialog
  2. Select "Save as PDF" as the printer
  3. Save the file as "YourStartupName-DPIIT-Certificate.pdf"
  4. Use this PDF to test the verification upload system
